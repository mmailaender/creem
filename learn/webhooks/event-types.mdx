## subscription.paid

The payment was successful, Creem collected the payment and started a subscription with this specific customer.

<AccordionGroup>
  <Accordion title="Sample Request Body">
  ```json
    {
        "type": "subscription.paid",
        "subscription_id": "sub_123456789",
        "customer_id": "cus_123456789",
        "product_id": "prod_123456789",
    }
  ```
    
  </Accordion>
</AccordionGroup>
